# TaskLogger

Веб-приложение для учёта рабочих задач, совместной работы по проектам и формирования отчётов. Разработано для государственных и офисных структур.

**Стек:** Django 4.2 · SQLite · Chart.js 4 · SortableJS · emoji-mart v5 · Vanilla JS

---

## Быстрый старт

```bash
git clone https://github.com/Maniloc/TaskLogger.git
cd TaskLogger

python -m venv venv
source venv/bin/activate        # Linux / Mac
venv\Scripts\activate           # Windows

pip install -r requirements.txt

cp .env.example .env            # заполните SECRET_KEY и TIME_ZONE

python manage.py migrate
python manage.py createsuperuser
python manage.py runserver
```

Откройте в браузере: **http://127.0.0.1:8000**

## Возможности

### Проекты и задачи

- Создание проектов с указанием инициатора и обоснования
- Задачи: дата, описание, статус, часы, инициатор, обоснование, даты начала и срока
- Статусы: Выполнено · В работе · Запланировано · Отложено
- Умный ввод часов: `2.5` · `2,5` · `2ч30м` · `2:30`
- Быстрое добавление задачи из любой страницы (кнопка «+ задача» в навбаре)
- Фильтрация по тексту, месяцу, статусу; пагинация (25 задач/страница)
- Назначение задачи конкретному исполнителю

### Участники проекта

Совместная работа в стиле Битрикс24:

| Роль | Права |
|------|-------|
| **Владелец** | Полный контроль, управляет составом, видит всё в отчётах |
| **Исполнитель** | Создаёт задачи, видит все, редактирует только свои |
| **Наблюдатель** | Только просмотр задач и отчётов |

- Страница `/projects/<id>/members/` — добавление и управление участниками
- При назначении задачи — автоматическое уведомление в чат
- Раздел **«Мои задачи»** — персональный дашборд по всем совместным проектам
- Отчёт: владелец видит всех, участник — только свои задачи

### Отчёты

- **Конструктор полей** — включайте/выключайте любые колонки одним кликом
  - Обязательные: название проекта, обоснование проекта, инициатор проекта, дата, часы, задача
  - По выбору: статус, обоснование задачи, инициатор задачи, дата начала, срок, исполнитель
- Текстовый вид с группировкой по проектам (удобен для вставки в документы)
- Таблица: задачи с подзаголовками проектов
- Экспорт в **Excel (.xlsx)**: только выбранные колонки, строки-заголовки проектов (синий фон), цветовая индикация статусов, итоговая строка
- Набор полей сохраняется в URL

### Аналитика

- Динамика задач и часов по месяцам (12 мес.)
- Активность по дням — heatmap (30 дней)
- Распределение статусов (donut chart)
- Топ проектов по часам
- Просрочено по проектам
- Все виджеты реагируют на фильтр периода; быстрые пресеты: 7д · 30д · 90д · год

### Чат

Полнофункциональный мессенджер внутри приложения:

**Диалоги и организация**
- Личные DM и групповые беседы (любое число участников)
- **🔖 Избранное** — личный блокнот, всегда первый в списке
- Закрепление диалогов (📌 жёлтая полоса)
- Ручная сортировка drag & drop (порядок сохраняется в БД)
- Очистка истории и удаление диалога

**Функции сообщений (правая кнопка мыши)**
- ↩️ Ответить с цитатой
- ➡️ Переслать в другой диалог
- 📋 Копировать текст
- ✏️ Редактировать (помечается «изменено»)
- 🗑️ Удалить

**Медиа**
- Файлы до 10 МБ: изображения, документы, архивы
- Inline-просмотр с лайтбоксом
- Ctrl+V вставка из буфера, drag & drop
- Emoji-picker с поиском (emoji-mart v5)

**Статусы и уведомления**
- ✓ доставлено / ✓✓ прочитано (галочки в реальном времени)
- Онлайн-статус: 🟢 онлайн или «был(а) ДД.ММ.ГГГГ в ЧЧ:ММ МСК»
- Звуковое уведомление (Web Audio API)
- Мьют per-conversation

**Прочее**
- Поиск по сообщениям
- Индикатор «печатает...»
- Счётчик символов
- Метка «изменено» с временем правки
- Кликабельный аватар → профиль пользователя

### Профиль и пользователи

- ФИО, должность, отдел, email
- Фото или цветной аватар (6 цветов)
- Смена пароля
- Публичный профиль: онлайн-статус, статистика, кнопка «написать»
- Система приглашений: одноразовые ссылки для регистрации коллег

### Дашборд

- Статистика текущего месяца (задачи, часы, выполнено, просрочено)
- График активности за 14 дней
- Мини-диаграмма часов по проектам
- Напоминание при отсутствии записей 2+ дней

### Интерфейс

- **6 тем**: 4 тёмных (Slate, Forest, Crimson, Neutral) + 2 светлых (Paper, Warm)
- Переключатель в навбаре, выбор сохраняется в браузере
- Встроенная справка (7 вкладок, кнопка «?» в навбаре)

### Администрирование (суперпользователь)

- Список всех пользователей с аватарами и статистикой
- Блокировка/активация, сброс пароля, удаление аккаунта
- Задачи всех пользователей с фильтрами
- Управление приглашениями

## Настройка (.env)

```env
DEBUG=True
SECRET_KEY=замените-на-длинную-случайную-строку
ALLOWED_HOSTS=localhost,127.0.0.1
TIME_ZONE=Europe/Moscow
```

---

## Деплой на сервер

```bash
# .env для продакшна
DEBUG=False
SECRET_KEY=длинный-случайный-ключ-50+-символов
ALLOWED_HOSTS=yourdomain.com

python manage.py collectstatic
pip install gunicorn
gunicorn tasktracker.wsgi:application --bind 0.0.0.0:8000
```

Рекомендуется Nginx как прокси перед gunicorn.
